# Simple Examples with redux

## Installation
---
```bash
npm install
or
yarn install
```

##### If - iOS

```bash
cd ios
pod install
```


## Run
---
##### iOS

```bash
yarn android
or
npm run android
```

##### Android
---
```bash
yarn android
or
npm run android
```

### Technologies used

* [ReactNative]
* [Axios]
* [ReactNavigation]
* [Redux]
* [ReduxThunk]
